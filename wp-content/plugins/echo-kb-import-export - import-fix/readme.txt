Import KB Articles and KB Categories.


== Description ==

The Import utility will allow you to import KB articles and KB Categories into your Knowledge Base.


== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-kb-import-export.zip.
2. Unzip the file echo-kb-import-export.zip.
3. Upload the 'echo-kb-import-export' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=add-ons
